Detailed Description

core_string_tの内部構造体。文字列バッファとメタ情報を保持する。

この構造体は core_string_t の実装における内部状態を表す。 利用者が直接この構造体にアクセスすることは想定されておらず、 core_string.c内でのみ使用される。
